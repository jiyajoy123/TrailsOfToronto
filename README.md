# Trails of Toronto

## Overview
**Trails of Toronto** is an Android application developed in Java using Android Studio. The app serves as a comprehensive travel guide for exploring Toronto, featuring various attractions, hotels, and a city map to enhance the user experience. Users can also maintain a favorites list and make notes about their favorite spots.

## Features
- **View Attractions**: Discover popular tourist attractions in Toronto, complete with descriptions and images.
- **Hotel Listings**: Explore various hotels, their amenities, and booking options.
- **City Map**: Navigate through Toronto with an integrated city map to find attractions and hotels easily.
- **Favorites List**: Save your favorite attractions and hotels for quick access later.
- **Notes**: Make personal notes about places you visit or plan to visit.

## Technology Stack
- **Language**: Java
- **Development Environment**: Android Studio
- **Database**: SQLite
